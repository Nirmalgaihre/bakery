## 📖 Overview
**Deurali Chemicals** is a high-performance web application built to supersede traditional desktop-based accounting software like Tally. This system offers a seamless, secure, and rapid workflow for inventory management, point-of-sale (POS) operations, and detailed financial reporting.

## 🚀 Key Technical Pillars
*   **Robust Framework:** Engineered with **Laravel** for scalability and clean architecture.
*   **Secure Authentication:** Multi-layered security protocols featuring custom **OTP Verification** via **PHPMailer** SMTP integration.
*   **Database Reliability:** Optimized **SQL/MySQL** relational database design to handle complex inventory and billing transactions.
*   **Professional UI/UX:** A responsive, mobile-optimized dashboard styled with **Tailwind CSS**.

## 🔒 Security Infrastructure
We prioritize data integrity and account security. The system includes:
- **Encrypted Authentication:** Hashed credentials and secure session management.
- **2FA Verification:** Real-time OTP generation and verification for every login attempt.
- **Brute-Force Protection:** Intelligent rate-limiting on all login and password reset routes.
- **Tokenized Recovery:** Secure, expiring, and hash-verified links for account recovery.

## 🛠 Features
- **Centralized Inventory:** Real-time stock monitoring and batch tracking.
- **Automated Billing:** Instant invoice generation with print-ready templates.
- **Intelligent Reporting:** On-demand data analytics for better decision-making.
- **Release Tracking:** Built-in changelog system for transparency on system updates.
